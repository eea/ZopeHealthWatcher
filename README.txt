=================
ZopeHealthWatcher
=================

ZopeHealthWatcher allows you to monitor a Zeo installation
installed on an unix system through a console script.

Installation
============

- Install the provided `DeadlockDebugger` in your Zope
  installation, following the instruction here:
     http://plone.org/products/deadlockdebugger

- Install the health watcher by running the setup script::

    $ python setup.py install
